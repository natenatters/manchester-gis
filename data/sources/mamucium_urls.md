## API Endpoints

### Pleiades

```
GET https://pleiades.stoa.org/places/79590/json
```

Returns GeoJSON with fort boundary, coordinates, and metadata.

### Historic England (requires token for some queries)

```
Base: https://services-eu1.arcgis.com/ZOdPfBS3aqqDYPUQ/ArcGIS/rest/services/
Scheduled Monuments: Scheduled_Monuments/FeatureServer/0/query
```

### Wikidata SPARQL

```
Endpoint: https://query.wikidata.org/sparql
Query for Mamucium: SELECT * WHERE { wd:Q3285089 ?p ?o }
```
